Codeforces上的一些简单问题

Abstract:一些有意思的Codeforces 800-1100难度有意思的小问题

Apples in Boxes

题目链接：text url
题意：给定长度为$ n $的数列$a$，Tom和Jerry轮流进行游戏，Tom先手，每次选择一个$a_i>0$，使得 $a_i$ 减小 $1$。
若没有$a_i>0$则当前操作者输，若操作后数列极差小于$k$，操作者输。计算谁会赢。
两人操作均为理智的。
思路：为了避免极差大于$k$使得极差尽可能小，最优选择是将数列最大项减$1$，极差单调递减。
设极差为$R$，若一开始$R-1>k$，即Tom操作一次就输。
或是若$R>k$且有多个相等的最大项，则Tom必输。
或是若无论如何$k>=R$，那么当数列之和为偶数时，Tom必输。

#include<bits/stdc++.h>
using namespace std;
#define int long long
#define all(x) x.begin(),x.end()
#define Yes cout<<"Yes\n"
#define No cout<<"No\n"
const int N=2e5+10;
void solve(){
    int n,k;
    cin>>n>>k;
    vector<int> a(n+1);
    for(int i=1;i<=n;i++){
        cin>>a[i];
    }
    int max=*max_element(a.begin()+1,a.end());
    int min=*min_element(a.begin()+1,a.end());
    int cnt = std::ranges::count(a, max);
    int sum=accumulate(a.begin(),a.end(),0ll);
    if (max - 1 - min > k || (max - min > k && cnt > 1) || sum % 2 == 0) {
    	cout << "Jerry\n";
    } else {
    	cout << "Tom\n";
    }
}
signed main(){
    ios::sync_with_stdio(false);

    cin.tie(nullptr);
    cout.tie(nullptr);
    int T=1;
    cin>>T;
    while(T--) solve();
    return 0;
}
