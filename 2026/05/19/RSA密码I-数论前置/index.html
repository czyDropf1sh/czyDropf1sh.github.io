RSA密码I:数论前置

Posted on 2026-05-19

RSA密码I:数论前置

定理 3.1（关于 $pq$ 的欧拉公式）**.** 设 $p$ 和 $q$ 为互异素数，令

$$g = \gcd(p-1, q-1).$$

则

$$a^{(p-1)(q-1)/g} \equiv 1 \pmod{pq}$$

对所有满足 $\gcd(a, pq) = 1$ 的 $a$ 成立。

特别地，若 $p$ 和 $q$ 为奇素数，则

$$a^{(p-1)(q-1)/2} \equiv 1 \pmod{pq}$$

对所有满足 $\gcd(a, pq) = 1$ 的 a 成立。

证明. 由假设可知 p 不整除 a，且 g 整除 q − 1，因此可以计算

$$a^{(p-1)(q-1)/g} = \left(a^{(p-1)}\right)^{(q-1)/g}$$

由于 $(q-1)/g$ 是整数，

$$\equiv 1^{(q-1)/g} \pmod{p} \quad \text{因为 } a^{p-1} \equiv 1 \pmod{p}$$

根据费马小定理，
$$\equiv 1 \pmod{p} \quad \text{因为 1 的任意次幂都是 1!}$$

将 $p$ 和 $q$ 的角色互换，完全相同的计算表明

$$a^{(p-1)(q-1)/g} \equiv 1 \pmod{q}.$$

这证明了 $a^{(p-1)(q-1)/g} − 1$ 同时被 $p$ 和 $q$ 整除；因此它被 $pq$ 整除，这就完成了定理 3.1 的证明。

命题 3.2. 设 p 为素数， e ≥ 1 为满足 gcd(e, p−1) = 1 的整数。命题 1.13 告诉我们， e 在模 p − 1 下有逆元，设为$de \equiv 1 \pmod{p-1}$则同余式$x^e \equiv c \pmod{p} $具有唯一解 $x ≡ c^d \pmod{p}$.

证明. 若 $c ≡ 0 (mod p)，则 x ≡ 0 (mod p)$ 是唯一解，证毕。因此假设 $c ≡ 0 (mod p)$。证明是费马小定理（定理 1.24）的一个简单应用。同余式 $de ≡ 1 (mod p − 1)$ 意味着存在整数 $k$ 使得

$$de = 1 + k(p-1).$$

现在验证 $c^d$ 是 $x^e \equiv c \pmod{p}$ 的解：

$$(c^d)^e \equiv c^{de} \pmod{p}$$
指数法则，

$$\equiv c^{1+k(p-1)} \pmod{p}$$
$$\equiv c \cdot (c^{p-1})^k \pmod{p}$$
$$\equiv c \cdot 1^k \pmod{p}$$
根据费马小定理，
$$\equiv c \pmod{p}.$$

这完成了 $x = c^d$ 是 $x^e \equiv c \pmod{p}$ 的解的证明。

为证唯一性，假设 $x_1$ 和 $x_2$ 都是同余式 (3.2) 的解。我们刚刚证明了对于任意非零值 z，$z^{de} \equiv z \pmod{p}$，因此

$$x_1 \equiv x_1^{de} \equiv (x_1^e)^d \equiv c^d \equiv (x_2^e)^d \equiv x_2^{de} \equiv x_2 \pmod{p}.$$

因此 $x_1$ 和 $x_2$ 在模 p 意义下相同，故 (3.2) 至多有一个解。$\square$

例 3.3. 求解同余式

$$x^{1583} \equiv 4714 \pmod{7919}$$

其中模数 p=7919 是素数。命题 3.2 指出，首先需要求解同余式

$$1583d \equiv 1 \pmod{7918}$$

使用扩展欧几里得算法（定理 1.11；也可参见注记 1.15 和练习 1.12），解为 $d \equiv 5277 \pmod{7918}$。然后命题 3.2 告诉我们

$$x \equiv 4714^{5277} \equiv 6059 \pmod{7919}$$

是 $x^{1583} \equiv 4714 \pmod{7919}$ 的解。

注记 3.4. 命题 3.2 包含假设 $$gcd(e, p − 1) = 1$$。如果省略该假设，则同余式 $x^e \equiv c \pmod{p}$ 对某些 $c$ 值有解，但不是对所有 $c$ 值都有解。此外，如果有解，则将有多个解。。

命题 3.5. 设 p 和 q 为互异素数，设 $e \geq 1$ 满足

$$\gcd(e, (p-1)(q-1)) = 1.$$

命题 1.13 告诉我们， $e$ 在模 $$(p − 1)(q − 1)$$ 下有逆元，设为

$de \equiv 1 \pmod{(p-1)(q-1)}.$

则同余式

$x^e \equiv c \pmod{pq} $

具有唯一解 $x ≡ c^d \pmod{pq}$.

证明. 假设 gcd(c, pq) = 1；其他情况见练习 3.3。命题 3.5 的证明与命题 3.2 的证明几乎相同，但不是使用费马小定理，而是使用欧拉公式（定理 3.1）。同余式 $$de \equiv 1 (mod (p − 1)(q − 1))$$ 意味着存在整数 k 使得

$$de = 1 + k(p-1)(q-1).$$

现在验证 $c^d$ 是 $x^e \equiv c \pmod{pq}$ 的解：

$$(c^d)^e \equiv c^{de} \pmod{pq} \qquad \text{指数法则，}$$

$$\equiv c^{1+k(p-1)(q-1)} \pmod{pq} \qquad \text{因为 } de = 1+k(p-1)(q-1),$$

$$\equiv c \cdot (c^{(p-1)(q-1)})^k \pmod{pq} \qquad \text{再次应用指数法则，}$$

$$\equiv c \cdot 1^k \pmod{pq} \qquad \text{根据欧拉公式（定理 3.1），}$$

$$\equiv c \pmod{pq}.$$

这完成了 $x = c^d$ 是同余式 (3.3) 的解的证明。还需证明解是唯一的。假设 $x = u$ 是 (3.3) 的解。则

$$(c^d)^e \equiv c^{de} \pmod{pq} $$
根据指数法则

$$\equiv c^{1+k(p-1)(q-1)} \pmod{pq}$$
因为de = 1 + k(p-1)(q-1),
$$\equiv c\cdot \big(c^{(p-1)(q-1)}\big)^k \pmod{pq}$$
再次应用指数法则
$$\equiv c\cdot 1^k \pmod{pq}$$
根据欧拉公式
$$\equiv c \pmod{pq}.$$

因此 (3.3) 的每个解都等于 $c^d \pmod{pq}$，故这是唯一解。

注记 3.6*.* 命题 3.5 给出了求解 $$x^e \equiv c \pmod{pq}$$ 的算法，该算法首先求解 $de ≡ 1 \pmod{(p − 1)(q − 1)}$，然后计算 $c^d \pmod{pq}$。使用一个更小的 d 值通常可以加速计算。设 $$g = gcd(p − 1, q − 1)$$，假设我们求解下列关于 $d$ 的同余式：

$$de \equiv 1 \pmod{\frac{(p-1)(q-1)}{g}}.$$

欧拉公式（定理 3.1）表明 $$a^{(p−1)(q−1)/g} \equiv 1 (mod pq)$$。因此，正如命题 3.5 的证明，如果记 $de = 1 + k[(p − 1)(q − 1)/g]$，那么

$$(c^d)^e = c^{de} = c^{1+k(p-1)(q-1)/g} = c \cdot (c^{(p-1)(q-1)/g})^k \equiv c \pmod{pq}.$$

因此使用此更小的 $d$ 值，我们仍然得出 $c^d \pmod{pq}$ 是 $x^e ≡ c \pmod {pq}$ 的解。
